Clean, consistent PixInsight workspaces.

Chapel Astro Utilities batch-processes open PixInsight images: turn long stacked master frame names into readable identifiers, save batches deliberately, and line up open image windows to a chosen reference.

Built for the messy middle of image processing.

PixInsight can generate beautifully calibrated data, wildly long window names, and image windows that all open at different sizes. These utilities keep the workspace readable when you are moving fast.

1

Line up open images

Use one image as the reference, then batch-match the other open windows to its zoom, frame size, and pixel dimensions when needed.

2

Reduce naming friction

Batch-rename stacked masters such as filter-tagged luminance, red, hydrogen, or OSC frames into clean identifiers that are easy to scan in the left image tabs.

3

Track processing stages

Batch-append suffixes such as PreNoiseX, PostCrop, or PreStretch without rebuilding mappings or touching unrelated images.

4

Save with confidence

Choose between batch rename-only, save-as output, and overwrite saves to the selected folder.

Recommended workflow order.

Start by making the open windows comparable, then batch-clean the names, batch-mark processing stages, and handle any one-off corrections.

1. Resize images to reference

Select a reference image and batch-match the other open windows to its zoom and frame. If their pixel dimensions differ, the utility resamples them to the reference dimensions.

2. Shorten long file names

Match words such as Red, Hydrogen, or Nofilter in the file path, caption, or image ID and batch-rename them to concise identifiers such as R, H, or OSC.

3. Append file name suffixes

After a crop, gradient correction, or noise reduction pass, batch-append a PixInsight-safe suffix to the currently open names. The filter mapping can be ignored when the current name is already the truth.

4. Rename individual image

Need to correct just one image? Select a single preview row, enter a valid identifier, and rename the side tab and top caption without running a batch operation.

What each utility does.

Both scripts live under the same PixInsight menu, but they solve different workspace problems.

ImageRenameByFilter

A batch naming and save workflow for open image windows. It is especially useful straight after stacking, when master frames inherit long file names from the integration process.

  • Batch-maps filename, caption, or view-id text to short filter names.
  • Stores editable user mappings between sessions.
  • Batch-appends PixInsight-safe processing suffixes to current names.
  • Renames one selected image with an ad-hoc identifier.
  • Saves renamed images, reopens saved output, and can close or collapse sources.
  • Overwrites selected files using the current open image name and selected folder.

MatchOpenImageSizes

A batch visual consistency tool for comparing open images. Choose the image that already looks right, then bring the rest of the workspace into line.

  • Uses the selected image window as the reference.
  • Copies the reference zoom factor to all other open images.
  • Copies the reference window frame size when PixInsight exposes writable geometry.
  • Resamples images whose pixel dimensions differ from the reference.
  • Leaves the reference image unchanged.
  • Supports PixInsight process icons for repeated reference-based use.
  • Can open ImageRenameByFilter automatically after the matching batch completes.

Install through PixInsight updates.

Add the repository once, then update from PixInsight whenever a new Chapel Astro Utilities package is published.

Repository URL

https://raw.githubusercontent.com/GrandPaClanger/pixinsight-toolbox-scripts/main/
  1. Open Resources > Updates > Manage Repositories.
  2. Add the repository URL above.
  3. Run Resources > Updates > Check for Updates.
  4. After installation, use Script > Chapel Astro Utilities.

User guide.

The short version: choose the images, choose the operation, check the preview, then apply or save.

ImageRenameByFilter

  • Choose a mode: select either Rename By Filter Mappings or Append suffix to current names. Apply stays disabled until a mode is selected.
  • Edit mappings: add, update, or delete rows in the mapping table. More specific matches win before generic ones.
  • Use valid names: output identifiers must start with a letter or underscore and only contain letters, digits, or underscores.
  • Select the batch: checked preview rows are processed. Collapsed windows are unchecked by default when PixInsight exposes that state.
  • Apply Rename / Append: batch-renames open images. If saving is not enabled, images are not closed, collapsed, or saved.
  • Save images after renaming: saves the renamed output to the selected folder and can reopen saved images.
  • After save: leave, close, or collapse selected source images.
  • Single Image Rename: select exactly one preview row, enter an ad-hoc name, and rename both the side tab and top caption.
  • Save & Overwrite Selected: saves checked open images using their current open image names in the selected folder.

MatchOpenImageSizes

Use this companion utility when open images need to match a selected reference image. It copies the reference zoom and attempts to match the frame size where PixInsight exposes that geometry.

  • Pick the reference: click the image window with the size, zoom, and dimensions you want to keep.
  • Run the utility: open Script > Chapel Astro Utilities > MatchOpenImageSizes.
  • Confirm the summary: the script reports the reference image, pixel size, zoom, and frame size before making changes.
  • Let it match the rest: every other open image is batch-matched to the reference zoom and frame size.
  • Understand resampling: if another image has different pixel dimensions, it is resampled to the reference width and height.
  • Continue into naming: enable Open ImageRenameByFilter after matching to launch the rename wizard as soon as the matching batch finishes.
  • Use process icons: drag the New Instance triangle to the workspace, then drag that process icon onto an image to use it as the reference.
  • Know the limit: if PixInsight does not expose writable window geometry, the script still matches zoom and dimensions but may not be able to copy the frame size.

Designed to protect the session you are in.

These scripts are intentionally direct, but they include guardrails for the naming and saving decisions that are easiest to get wrong late at night.

Identifier checks Ad-hoc names must follow PixInsight identifier rules and cannot duplicate another open image ID.
Selection preview The preview shows current, matched, new, and status columns before batch operations run.
Overwrite confirmation Overwrite operations ask once for the batch, then save selected images deliberately.
Folder defaults The save folder defaults to the source directory used by the majority of open images.
Reference preserved MatchOpenImageSizes leaves the selected reference image unchanged.
Resample warning Images with different pixel dimensions are resized to the reference, so choose the reference deliberately.
