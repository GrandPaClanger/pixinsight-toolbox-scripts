Built for the messy middle of image processing.

PixInsight can generate beautifully calibrated data and wildly long window names. These utilities keep the workspace readable when you are moving fast.

Three common workflows, one compact tool.

ImageRenameByFilter is designed around the real sequence most imagers follow: stack, clean up names, process, mark processing stages, and save.

Rename by filter mapping

Match words such as Red, Hydrogen, or Nofilter in the file path, caption, or image ID and rename them to concise identifiers such as R, H, or OSC.

Append a processing suffix

After a crop, gradient correction, or noise reduction pass, append a PixInsight-safe suffix to the currently open names. The filter mapping can be ignored when the current name is already the truth.

Rename one image ad hoc

Need to correct just one image? Select a single preview row, enter a valid identifier, and rename the side tab and top caption without running a batch operation.

Install through PixInsight updates.

Add the repository once, then update from PixInsight whenever a new Chapel Astro Utilities package is published.

Repository URL

https://raw.githubusercontent.com/GrandPaClanger/pixinsight-toolbox-scripts/main/
  1. Open Resources > Updates > Manage Repositories.
  2. Add the repository URL above.
  3. Run Resources > Updates > Check for Updates.
  4. After installation, use Script > Chapel Astro Utilities.

User guide.

The short version: choose the images, choose the operation, check the preview, then apply or save.

ImageRenameByFilter

  • Rename By Filter Mappings: best for newly stacked masters with long generated names.
  • Append suffix to current names: best after processing when the current names are already correct.
  • Single Image Rename: best for one-off corrections.
  • Save images after renaming: saves the renamed output to the selected folder.
  • Save & Overwrite Selected: saves selected images using their current open image names.

MatchOpenImageSizes

Use this companion utility when open images need to match a selected reference image. It copies the reference zoom and attempts to match the frame size where PixInsight exposes that geometry.

  • Select the reference image window.
  • Run Script > Chapel Astro Utilities > MatchOpenImageSizes.
  • Use a process icon when you want to apply the same reference workflow repeatedly.

Designed to protect the session you are in.

These scripts are intentionally direct, but they include guardrails for the naming and saving decisions that are easiest to get wrong late at night.

Identifier checks Ad-hoc names must follow PixInsight identifier rules and cannot duplicate another open image ID.
Selection preview The preview shows current, matched, new, and status columns before batch operations run.
Overwrite confirmation Overwrite operations ask once for the batch, then save selected images deliberately.
Folder defaults The save folder defaults to the source directory used by the majority of open images.
